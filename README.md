Setup
- Clone the repository:
git clone https://hirendra-creater.github.io/Search/ 
cd search-app  
- Install dependencies:
pip install -r requirements.txt  
- Run the app:
flask run  
- Access via browser at https://hirendra-creater.github.io/Search/
